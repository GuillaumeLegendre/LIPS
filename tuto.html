Configuration d'un langage

Ce tutoriel a pour objectif la définition et la configuration d'un langage de programmation. Nous dévelepperons donc en plusieurs parties :

  1. Se rendre dans la partie adéquate de l'administration
  2. Configurer le langage de programmation
  3. Configurer l'image de représentation du langage
  4. Configurer la base du code

3. Configurer l'image de représentation du langage


Image 5

1 ère étape :
Cette étape est facultative, elle permet de définir l'image de représentation d'un langage. Cette image sera utilisée notamment dans la page concernant le classement d'un étudiant en fonction des langages des cours auxquels il a pris part.
Le champ prévu pour l'image doit être rempli avec un lien vers une image à l'un des formats suivants : jpg, jpeg, png, gif, ico, bmp

2 ème étape :
Une fois le lien vers l'image renseigné, il suffit de cliquer surle bouton "Modifier" pour valider le changement d'image.

4. Configurer la base du code


Image 6

1 ère étape :
La balise <lips-preconfig-import> représente la zone où seront placés les imports définis dans les problème

2 ème étape :
La balise <lips-preconfig-code> représente la zone où sera placé le code écrit par l'étudiant résolvant le problème

3 ème étape :
La balise <lips-preconfig-test> représente la zone où seront placés les test unitaires fournis par les professeurs afin de valider les implémentation des élèves

Remarque : L'utilisation des trois balises présentées ci-dessus n'est pas optionnelle
