Résolution d'un problème

Ce tutoriel a pour objectif la résolution d'un problème. Nous dévelepperons donc en plusieurs parties :

  1. Résoudre le problème
  2. Défier un autre membre
  3. Afficher les solutions du problème
  4. Modifier/Supprimer un problème


Image 1

1. Résoudre le problème

Pour résoudre le problème il suffit de fournir son implémentation dans le cadre "Réponse" (Flèche 1)

2. Défier un autre membre

Pour défier un autre membre il faut cliquer sur le bouton "Défier" (Flèche 2), une fenêtre s'ouvre alors (Image 2). Dans cette fenêtre vous pourrez taper dans un champ avec autocomplétion les noms des membres que vous souhaitez défier


Image 2

3. Afficher les solutions du problème

En cliquant sur le bouton "Solutions", vous avez la possibilité de voir les solutions (Image 3) qui ont déjà proposées pour le problème. Attention, le bouton "Solutions" s'affiche uniquement si vous avez déjà fourni une solution au problème ou bien si vous avez créer le problème


Image 3

4. Modifier/Supprimer un problème

Si vous êtes le créateur du problème vous avez également la possibilité de modifier ou de supprimer le problème en cliquant respectivement sur les boutons "Modifer" et "Supprimer" (Flèches 4 et 5)
